Track Your Crypto & Fiat Finances with Airtable

Since I started owning crypto-currencies, I’ve had a harder and harder time keeping track of my finances. With my modest fortune spread between multiple bank accounts, ether and bitcoin addresses, I needed a better way to know my total net worth and what the relative values were of my assets. I decided to use Airtable and its extensive API to build a solution that could do just that.

In this post I will show you how you too can use Airtable and a simple web server (finance-tracker-server) to keep track of your finances. It will update your bank balances, pull in the latest crypto and fiat currency exchange rates in order to give you up-to-the-minute profit & loss calculations, total net worth valuation, and where you lie on the crypto/fiat split. I will walk you through customizing the Finance Tracker Airtable Base I’ve published, and how to set up the server that will keep it up to date.

Embedded version of the Finance Tracker Base (Click around!)

The Finance Tracker base is made up of two tables. The “Assets” table has a record for each assets you own, be they bank balances or crypto-currencies you keep in an offline wallet/exchange. It reports how each asset is performing in US Dollar terms (although with the flexibility of Airtable, this could to be any currency you want).

The second table keeps track of the “Currencies” (both crypto and fiat) and their relative exchange prices in USD.

Screenshot of the Currencies Table

The finance-tracker-server will update the currency exchanges you care about in the “Currencies” table. Since every asset is denominated in a currency, these updates will propogate to the “Assets” table, updating the USD value of your assets given the new exchange rates. In addition to this, it will also update your bank balances directly in the “Assets” table so that any expenses you’ve had since the last update will be reflected here.

Getting Started

In order to get started, you will need to create an Airtable account and install the Finance Tracker base onto one of your teams. You can do this by clicking the “Copy Base” button in the bottom right corner of the embedded base in this post. This will install your very own copy of the “Finance Tracker” base which you can then customize to represent your own finances.

The “Finance Tracker” base comes pre-filled with dummy data in the “Assets” table. Each record in this table should represent an asset you own. You can delete all the dummy entries by selecting all the records (cmd-a) and right clicking to see the “Delete all selected records” option. This will make room for your actual assets, be they crypto or fiat.

The first thing you should do is populate this table with each of the assets you own, filling in the “Funds” field with a descriptive name for the asset, the “Amount” field with the amount of the asset you own and the “Currency” field with the currency the asset is denominated in. You should also fill in the “Unit Buy Price” you paid for the asset (this is simply $1 if it’s a USD denominated asset).

Pro tip: If you bought a crypto-currency at multiple times at different prices, you can create a record for each time you bought or mined the currency. This way, you can more accurately track your gains/losses for each subsequent acquisition.

The rest of the fields in this table will be auto-calculated for you based on the other field values and the currency exchange rates in the “Currencies” table. This is because they are formula fields that take as input the values of other fields. For example, the “Gain/Loss” field is simply the “Current Value” field minus the “Total Buy Price” field.

Formula editor

Pro tip: You can inspect and edit each formula by double clicking the field name, opening up the field configuration menu.

Another Airtable feature used is the summary bar at the bottom of the base. This allows you to see aggregate stats about a particular column. For example, to see your total net worth, you can look at the summation at the bottom of the “Current Value” field:

The same goes for the “Crypto” and “Personal” columns which tell you the amount of your net worth that is in crypto assets and fiat assets.

Setting up the server

Now that you’ve listed all of your assets, it’s time to install and configure the server that’s going to update the currency exchange rates and update your bank balances whenever you want. To do this, go ahead and clone the finance-tracker-server and follow the detailed README instructions on how to customize which currencies you’d like updated (both crypto and fiat) as well as how to pull in your bank account balances into Airtable.

After you have configured and successfully updated your Finance Tracker app, continue from over here.

Ok great! You’ve now successfully customized your Airtable Finance Tracker base and can keep it up to date with a server running locally on your computer! The cool thing about keeping track of your finances in this way is that Airtable is extremely flexible. If there is an additional statistic or key metric you want to keep track of, you can easily add a formula field and calculate it right away! If you suddenly buy a new crypto-asset or open a bank account abroad, you can just add them to the assets table and start tracking them as well.

If you want to update your Finance Tracker on-the-go, you can deploy the finance-tracker-server to a remote server. This way you don’t need to run anything locally, you can simply visit a URL and immediately see your updated finances. Let me know if you’d like me to write up a tutorial on how to do this.

I hope you found this tutorial helpful and that it helps you stay on top of your finances!
